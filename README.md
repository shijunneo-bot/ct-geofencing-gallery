# CleverTap Geofencing · Field Guide 08

120 location-engagement plays across 15 verticals, plus the Developer Setup Docs, in one self-contained HTML file. No CDN, no external assets.

## Deploy to GitHub Pages

1. Create the repo `ct-geofencing-gallery` under the `shijunneo-bot` account (public).
2. Add `index.html` to the repo root on the `main` branch and push.
3. Repo → Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / `/ (root)` → Save.
4. Live in about a minute at: https://shijunneo-bot.github.io/ct-geofencing-gallery/
